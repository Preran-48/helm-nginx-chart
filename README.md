# Helm NGINX Chart

A hands-on project for learning and developing Kubernetes Helm charts using NGINX. This repository covers both understanding the default chart generated by Helm and building a custom Helm chart from scratch.

---

## Repository Structure

```
helm-nginx-chart/
│

│
├── reference-nginx-chart/
│   ├── Chart.yaml

│   ├── charts/
│   └── templates/
│
└── my-nginx-chart/
    ├── Chart.yaml
    ├── values.yaml
    └── templates/
```

# Phase 1 - Understanding the Generated Helm Chart ✅

The `reference-nginx-chart` directory contains the chart generated using:

```bash
helm create nginx-chart
```

This chart was used to understand the default Helm chart structure and how Helm templates work.
